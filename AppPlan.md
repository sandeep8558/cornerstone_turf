# Cornerstone Mobile App Plan
 - Cornerstone Turf is a turf booking app.
 - User can book slot on hourly basis
 - It also has subscription plans

## Technology
 - Flutter

## Database and Tables
 - User

## Booking and Subscription Plans
 - Monday, Tuesday, Wednesday, Thursday and Friday booking amount Rs.799/Hour
 - Saturday and Sundays Booking amount Rs.1000/Hour
 - Coupon and Discounts

## User Roles
 - Administrator
 - Manager
 - Client

## Authentication
 - Login
 - Register
 - Forgot Password
 - Login with google
 - Change Password
 - Profile Edit with delete account
 - Dashboard
